- `Assignment 0` — Password security, authentication factors, phishing, and password managers
- `Assignment 1` — Data breaches, deletion, encryption, hashing, salts, digital signatures, and cryptography fundamentals
- `Assignment 2` — Zero-day attacks, port scanning, worms, VPN, SSH, X.509, penetration testing, and DDoS-related concepts

- `Assignment 4` — Privacy, private browsing, cookies, Tor, browser fingerprinting, sandboxing, user-agent data, and web tracking
